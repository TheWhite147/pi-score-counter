# pi-score-counter
Raspberry Pi score counter for games

## GPIO Inputs
- Player 1 - Button A
- Player 1 - Button B
- Player 2 - Button A
- Player 2 - Button B
- Reset button

## Use of buttons
- Always
    - Reset button: Resets the system to the home page
- Before a game
    - Button A - Change player
    - Button B - Select player
- During a game
    - Button A - Increase score
    - Button B - Decrease score
